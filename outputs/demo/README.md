```bash
bash scripts/run_demo.sh
```

Uses `--preset enright_rods` (cluster splitting + orange reject labels).

## How to read the overlays

| Color | Meaning |
|---|---|
| **Green** solid | Classified **rod** |
| **Blue** solid | Classified **dot** (sphere-like) |
| **Orange** dashed | **Reject** — detected blob that did not pass rod/dot rules |

## Debug image

`*_segments_debug.png` shows **every segmented blob** with a number **before** classification.
If a rod is missing from the main overlay but has a number on the debug image, classification failed.
If it is missing from both, segmentation failed.
